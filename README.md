# 赛题介绍
## 一、赛事基本信息
- **赛事全称**：第二届甘肃省大学生数据挖掘挑战赛（2025年）
- **主办单位**：甘肃省教育厅
- **承办单位**：甘肃农业大学
- **协办单位**：广东泰迪智能科技股份有限公司
- **参赛对象**：甘肃省全日制本科生，3人组队，不可跨校/跨组别
- **赛事规模**：全省32所高校、795支本科队伍参赛
- **赛题方向**：**农业多模态数据挖掘（石榴病害智能识别与预测）**

---

## 二、赛题
**融合石榴果实时序图像数据和传感监测数据的智能化果实图像病害阶段识别与病害发展演变预测**

---

## 三、赛题背景与意义
甘肃是我国石榴主产区之一，石榴病害（如干腐病、褐斑病等）直接影响产量与品质。传统病害识别依赖人工经验，效率低、滞后性强；而单一图像数据或环境传感数据难以精准刻画病害演变规律。

本题聚焦**农业多模态数据融合**，旨在通过时序图像与物联网传感数据，构建病害阶段识别与演变预测模型，为果园精准防控提供技术支撑，助力智慧农业发展。

---

## 四、数据说明
提供**两类核心数据**，均来自规模化石榴园实测：
1. **石榴果实时序图像数据**
    - 包含健康、干腐病、褐斑病等**8类病害阶段图像**
    - 时序序列：同一果实不同生长/发病阶段的连续图像
    - 标注信息：病害类型、发病阶段（初期/中期/后期）
2. **物联网传感监测数据**
    - 环境因子：温度、湿度、光照、土壤含水率等**时序传感数据**
    - 关联记录：病害发生时间、位置、严重程度的历史记录
    - 时间维度：与图像数据**时间戳严格对齐**，实现多模态数据融合

---

## 任务说明
### 任务1：石榴病害阶段精准识别（分类任务）
- 基于图像数据，识别**8类病害类型+对应发病阶段**（初期/中期/后期）
- 要求：构建高精度图像分类模型，解决小样本、病害特征相似的识别难点

### 任务2：病害发展演变预测（时序预测任务）
- 融合**时序图像特征+环境传感数据**，预测未来7/15天病害严重程度演变趋势
- 要求：挖掘环境因子（温湿度等）与病害发展的关联规律，构建时序预测模型

### 任务3：环境传感数据与病害发生关联挖掘
- 输入：同步的环境时序数据（温湿度、光照等）+ 病害记录
- 输出：
  - 关键环境因子筛选（哪些因子与病害爆发/加重最相关）
  - 相关性模型/规则（如高湿+高温→干腐病风险↑）
- 考核：特征选择、时序相关性分析、可解释性。

### 任务4：多模态融合病害发展预测
- 输入：**时序图像特征 + 环境传感数据**（多模态融合）
- 输出：未来 **7天、15天** 的病害严重度演变预测曲线
- 考核：融合策略（早融合/晚融合/中间融合）、预测误差（MAE/RMSE）、对比单模态模型提升幅度。
